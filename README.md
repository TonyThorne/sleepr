# Overview

This is a Udemy course 'Nest.js Microservices: Build & Deploy a Scalable Backend' by Michael Guay.

[Course](https://www.udemy.com/course/nestjs-microservices-build-deploy-a-scaleable-backend/)

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```
